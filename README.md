# Overview

Taking inspiration from platforms like [Capterra](https://www.capterra.com/), a software comparison tool, the Prospect Research Toolkit aims to be a central repository for information about specialized fundraising products. Other goals include building out prospect dev and fundraising industry datasets, and documenting the progress in the [Prospect Dev Field Notes blog](https://prospectresearchtoolkit.github.io/ProspectResearchToolkit/)
